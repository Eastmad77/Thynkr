SPECS
- Portrait primary: 1080x1920 JPG (quality 85–92)
- Landscape optional: 1920x1080 (use same names with -land.jpg if needed)
- Color space: sRGB
- Filenames locked to manifest.json to avoid code changes later.

REPLACEMENT WORKFLOW
1) Drop new JPG with the exact filename to swap art.
2) No code changes needed; the app reads manifest.json on boot.
3) SW cache will update; toast will prompt user to refresh.

LICENSING
- Ensure rights to all imagery. Store attribution (if any) here.
